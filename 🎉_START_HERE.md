## 💻 **QUICK START GUIDE**

### **Step 1: Download**
Download the `maxzi-dashboard` folder from the outputs directory

### **Step 2: Install Dependencies**

```bash
# Backend (Python)
cd backend
pip install -r requirements.txt

# Frontend (Node.js)
cd frontend
npm install
```

### **Step 3: Start the Dashboard**

**macOS/Linux:**
```bash
./start.sh
```

**Windows:**
```batch
start.bat
```

**Or manually:**
```bash
# Terminal 1 - Backend
cd backend && python app.py

# Terminal 2 - Frontend
cd frontend && npm start
```

### **Step 4: Open Browser**

```
🌐 http://localhost:3000
```
